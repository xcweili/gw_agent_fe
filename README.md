# 招投标智能体平台前端项目

这是一个基于Vue 3 + Element Plus开发的招投标智能体平台前端应用。

## 功能特性

### 1. 首页功能
- **左右分栏布局**：左侧为智能体列表，右侧为对话区域
- **智能体列表**：显示所有已注册的智能体，支持选择和切换
- **对话功能**：
  - 支持文本输入和文件上传
  - 流式响应显示
  - think过程展示（可展开/收起）
  - 快捷问题快速发送
  - "开启新对话"功能

### 2. 智能体管理页面
- **智能体列表展示**：网格布局展示所有智能体卡片
- **创建智能体**：
  - 支持图标上传（默认使用SVG图标）
  - 必填信息：名称、简介、API地址、API Key
  - 可选信息：快捷问题列表
  - 表单验证和错误提示

### 3. 本地数据管理
- **本地存储**：智能体信息保存在localStorage中
- **Mock模式**：支持模拟对话数据验证功能
- **数据持久化**：刷新页面后数据不丢失

### 4. 智能体对话功能
- **OpenAI格式API**：兼容标准的OpenAI API调用格式
- **多轮对话**：支持上下文记忆
- **流式响应**：模拟流式数据展示效果
- **错误处理**：完善的错误提示和处理机制

## 安装和运行

- Node.js >= 16
- npm >= 8

```bash
npm install
```

### 开发模式
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```

## 使用说明

### 1. 创建智能体
1. 点击左侧"更多智能体"或顶部"创建智能体"
2. 填写智能体信息：
   - **名称**：智能体的显示名称
   - **简介**：智能体的功能描述
   - **API地址**：智能体后端服务地址
   - **API Key**：访问凭证
   - **快捷问题**：可选，配置后会显示在对话区域
3. 点击"创建"保存

### 2. 对话功能
1. 选择左侧智能体开始对话
2. 在输入框中输入问题或上传文件
3. 点击发送或按Enter键发送消息
4. 支持快捷问题快速发送
5. 点击"开启新对话"清空历史记录

### 3. Mock模式
- 在智能体列表页可以开启Mock模式
- 开启后会使用预设的模拟回答
- 用于测试和演示功能

## 技术栈

- **Vue 3**：前端框架
- **Element Plus**：UI组件库
- **Vue Router**：路由管理
- **Axios**：HTTP请求
- **Vite**：构建工具

## 开发规范

- 使用Vue 3 Composition API
- 组件化开发，遵循单一职责原则
- 使用Element Plus组件库保持界面一致性
- 本地数据存储，使用localStorage持久化
- 完善的错误处理和用户反馈

## 注意事项

1. **API配置**：实际使用时需要配置有效的智能体API地址和Key
2. **文件上传**：当前版本的文件上传功能仅在前端展示，实际使用需要后端支持
3. **数据存储**：智能体信息存储在浏览器本地，清除浏览器数据会丢失
4. **Mock模式**：用于功能演示，生产环境建议关闭
